# Animator

## Getting Started

<AnimatorExamples />

```jsx
import { Animator } from "@lottie-web/react";
import groovyWalkAnimation from "./groovyWalk.json";

const AnimationExample = () => {
	return <Animator animationData={groovyWalkAnimation} />;
};

export default AnimationExample;
```

## Props

### `animationData`
A JSON Object with the exported animation data
```yaml
Type: Object
Default: none
Required: ☑
```

### `loop`
Set it to true for infinite amount of loops, or pass a number for a specific amount of loops
```yaml
Type: Boolean | Number
Default: true
Required: ☐
```

### `autoplay`
If set to true, animation will play as soon as it's loaded
```yaml
Type: Boolean
Default: true
Required: ☐
```

### `initialSegment`
Expects an array of length 2. First value is the initial frame, second value is the final frame. If this is set, the animation will start at this position in time instead of the exported value from AE
```yaml
Type: Array
Default: none
Required: ☐
```

## Interaction methods
These methods are designed to give you more control over the Lottie animation, and fill in the gaps left by the props:
- play
- stop
- pause
- setSpeed
- goToAndPlay
- goToAndStop
- setDirection
- playSegments
- setSubframe
- destroy
- getDuration

### Calling the methods
To use the interaction methods listed above, pass a reference to the Animator in your component by using the ref prop (see the React documentation to learn more about [Ref](https://reactjs.org/docs/refs-and-the-dom.html) or [useRef](https://reactjs.org/docs/hooks-reference.html#useref) hook):
```jsx
import { Animator } from "@lottie-web/react";
import groovyWalkAnimation from "./groovyWalk.json";

const AnimationExample = () => {
    const ref = useRef();

	return <Animator ref={ref} animationData={groovyWalkAnimation} />;
};

export default AnimationExample;
```

You can then use the interaction methods like this:
```jsx
...
ref.current.pause();
...
```
